# Projects Images

Add your project screenshots here!

## How to Add Projects

1. **Add your project images** to this folder:
   - `project1.jpg`
   - `project2.jpg`
   - `project3.jpg`
   - `project4.jpg`
   - etc.

2. **Edit the projects array** in `src/components/Projects.jsx`:

```javascript
const projects = [
  {
    title: 'My Awesome Project',
    description: 'Description of what this project does',
    image: '/projects/project1.jpg',  // Your image filename
    link: 'https://github.com/your-username/project-repo',
    skills: ['Python', 'Flask', 'Security'],
  },
  // Add more projects...
]
```

## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 1200x800px (3:2 ratio)
- **File size**: Keep under 500KB for fast loading
- **Naming**: Use simple names like `project1.jpg`, `project2.jpg`

## Tips

- Use screenshots of your project's main interface
- Show the most impressive features
- Compress images before uploading (use tinypng.com)
- Make sure images are clear and professional
